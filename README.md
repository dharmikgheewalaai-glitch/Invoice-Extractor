# 📑 Invoice Extractor

A simple app to extract tabular data from invoice PDFs and export them to **CSV/Excel**.

---

## 🚀 Features
- Upload PDF invoices
- Extracts line items & tables
- Normalizes headers (e.g., `Qty` → `Quantity`, `Invoice #` → `Invoice No`)
- Download as CSV or Excel
